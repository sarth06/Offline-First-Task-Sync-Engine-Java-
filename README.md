# OfflineSyncEngine (Java)

## Overview
A user-friendly, offline-first task management system with manual sync support.

## Features
- Menu-driven console UI
- Offline task creation, update, delete
- Persistent storage
- Manual sync with status tracking
- Clean layered architecture

## How to Run
1. Open project root in IntelliJ
2. Mark `src` as Sources Root
3. Run `app/Main.java`
